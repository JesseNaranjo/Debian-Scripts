## htop options

Display options
- Tree View (F5)
- Shadow distribution path prefixes
- Highlight new and old processes

Header layout
- 2 columns 33/67
- Column 1
  - Hostname [Text]
  - Uptime [Text]
  - Task counter [Text]
  - Load average [Text]
- Column 2
  - CPUs (1/1) [Bar]
  - Memory [Bar]
  - Swap [Bar]
